PULL DOCKER IMAGE FOR OPENWEBUI
>> docker pull ghcr.io/open-webui/open-webui:v0.8.3

THEN RUN IT 

>> docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main

>> LOGIN IN OPENWEBUI
PULL MODEL IN OPENWEBUI
>> keep ollama container running
>> run OPENWEBUI container 
>> admin panel -> settings -> MODEL -> manage -> name a model and pull
